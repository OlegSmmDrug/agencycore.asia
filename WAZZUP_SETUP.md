## Настройка в Wazzup

1. Войдите в панель управления Wazzup
2. Перейдите в раздел "Webhooks"
3. Добавьте новый webhook с URL выше
4. Укажите события:
   - Новые сообщения (messages)
   - Статусы сообщений (statuses)

## Использование в системе

### Вкладка WhatsApp

Новая вкладка "WhatsApp" в сайдбаре предоставляет:

- **Список всех чатов** с клиентами из CRM
- **Последнее сообщение** и время
- **Счетчик непрочитанных** сообщений
- **Полная переписка** с каждым клиентом

### Чат в карточке клиента

В карточке клиента (CRM) также есть встроенный чат WhatsApp для быстрого общения.

## Как работает привязка

1. Когда поступает новый лид из Creatium, система создает клиента с номером телефона
2. Входящие сообщения WhatsApp автоматически привязываются к клиенту по номеру телефона
3. Менеджер может вести диалог из вкладки WhatsApp или из карточки клиента

## Проверка работы

После настройки webhook:

1. Отправьте тестовое сообщение на номер WhatsApp канала
2. Проверьте, что сообщение появилось во вкладке WhatsApp
3. Отправьте ответ из системы
4. Убедитесь, что клиент получил сообщение
